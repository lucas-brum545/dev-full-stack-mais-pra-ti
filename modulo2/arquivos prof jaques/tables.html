Tabelas HTML na prática

Tabela é para dados (linhas e colunas), não para diagramar a página. Para layout, use Grid ou Flexbox.

1) Estrutura semântica completa — thead, tbody e tfoot

Repare na divisão de papéis: cabeçalho, corpo de dados e rodapé com o total.
