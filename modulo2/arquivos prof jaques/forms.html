Formulários HTML na prática

Todo campo tem um <label> conectado. Tente enviar com campos vazios para ver a validação nativa do navegador.

1) Texto, e-mail e senha — type muda tudo

No celular, cada type abre um teclado diferente (arroba no e-mail, números no telefone).

Validado automaticamente: precisa conter "@".
Mínimo de 8 caracteres (atributo minlength).

2) Tipos especiais — number, date e range

Cada um traz um controle pronto: setinhas, calendário e um "slider".

3) select — escolher UM item de uma lista

Ideal quando há muitas opções (estados, países). Poucas opções? Prefira radio.

4) radio e checkbox — agrupados em fieldset

radio = escolha ÚNICA (mesmo name). checkbox = várias ao mesmo tempo.

Plano (escolha um)
Interesses (marque quantos quiser)

5) textarea — texto longo de várias linhas

Para mensagens, comentários, descrições. O <input> é só de uma linha.

Até 300 caracteres.

6) Enviar o formulário

type="submit" dispara a validação e o envio; type="reset" limpa tudo.
